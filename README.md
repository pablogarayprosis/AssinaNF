# AssinaNF

Para clientes do Affinco NF que usam certificado digital A3.

## Componentes

### RestClient

Utilizado para download dos xmls.

Disponível no servidor-prosis em E:\Desenvolvimento\Componentes\delphi-rest-client

Instalar o pacote para o Delphi XE3 que está em packages\XE3.
Adicionar no library path as pastas lib e src


### Alcinoe
Utilizado para comujnicação com a api.

Disponível no servidor-prosis em E:\Desenvolvimento\Componentes\alcinoe

Instalar o dpk que está na pasta source.
Adicionar no library path as pastas source e resource
